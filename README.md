# Gym-Progress-Tracker
A simple and structured workout session tracking application for the gym built using Object-Oriented Programming (OOP) principles
This simple app allows users to input workouts, track both sets and reps, and manage full workout sessions through a Command Line Interface (CLI) and Graphical User Interface (GUI)

Overview

This project models real-world gym workflows using OOP concepts. Users can:

- Create multiple workouts
- Track sets, reps, weight, and rest time
- Progress through a full workout session
- Use either CLI or a GUI interface

The goal of this project is to demonstrate strong foundational programming skills, particularly in class design, state management, and user interaction.

Features
1. Object-Oriented Design
- Workout class handles individual exercise logic
- Session class manages multiple workouts
- Clear separation of responsibilities
2. Workout Tracking
Track:
- Sets
- Reps
- Weight
- Rest time
Automatically updates progress after each set
3. Rest Timer
- Built-in countdown timer in GUI
- Displays remaining rest time dynamically
Dual Interface Support
1. CLI Mode
- Simple terminal-based interaction
2. GUI Mode (tkinter)
- Interactive and user-friendly interface
- Buttons for completing sets and progressing workouts
